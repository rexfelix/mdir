# mdir - Modern Mdir CLI

MS-DOS 시절 Mdir 3.x의 사용자 경험을 현대 터미널 환경에서 재현한 TUI 파일 관리자입니다.

## 스크린샷 (예정)

## 주요 기능

- **멀티 컬럼 파일 목록** - 터미널 너비에 따라 1/2/3 컬럼 자동 전환
- **키보드 중심 탐색** - 방향키로 파일 탐색, Enter/Backspace로 디렉토리 이동
- **파일 선택(마킹)** - Space 키로 복수 파일 선택, 선택 파일 노란색 표시
- **파일 CRUD** - 복사(C), 이동(M), 삭제(D), 이름변경(R), 새 폴더(K), 새 파일(N)
- **내부 뷰어** - V 키로 텍스트 파일 열람, 스크롤, 검색(/), 매치 탐색(n/N)
- **내부 에디터** - E 키로 텍스트 파일 편집, 저장(Ctrl+S), 멀티바이트(한글) 안전
- **파일 검색** - F 키로 현재 디렉토리 하위 재귀 검색, 와일드카드(*, ?) 지원
- **상태바** - 현재 경로, 선택 파일 정보, 파일/디렉토리 수 표시
- **숨김 파일 토글** - H 키로 숨김 파일 표시/숨김 전환
- **파일 타입별 색상** - 디렉토리(하늘), 실행파일(녹색), 압축파일(노랑), 심볼릭링크(보라)
- **정보창 고도화** - 파일 권한(rwxrwxrwx), 디스크 사용량 표시
- **안정적인 터미널 복원** - panic 발생 시에도 터미널 상태 복원

## 설치

### 요구사항

- Rust 1.85 이상
- macOS, Linux, Windows(WSL) 지원

### 빌드

```bash
git clone <repository-url>
cd mdir
cargo build --release
```

바이너리는 `target/release/mdir`에 생성됩니다.

### PATH에 추가 (선택)

```bash
cp target/release/mdir ~/.local/bin/
```

# 또는 처음부터

```bash
git clone <repository-url>
cd mdir
cargo install --path .
```

## 사용법

```bash
# 현재 디렉토리에서 실행
mdir

# 또는 cargo로 직접 실행
cargo run --release
```

### 단축키

| 키 | 기능 |
|---|------|
| `↑` `↓` `←` `→` | 커서 이동 (좌/우는 컬럼 간 이동) |
| `Enter` | 디렉토리 진입 |
| `Backspace` | 상위 디렉토리로 이동 |
| `Home` / `End` | 목록 처음/끝으로 이동 |
| `PageUp` / `PageDown` | 페이지 단위 이동 |
| `Space` | 파일 선택/해제 (복수 선택 가능) |
| `C` | 선택 파일 복사 |
| `M` | 선택 파일 이동 |
| `D` | 선택 파일 삭제 (확인 후) |
| `R` | 파일 이름 변경 |
| `K` | 새 디렉토리 생성 |
| `N` | 새 빈 파일 생성 |
| `V` | 파일 내용 보기 (내부 뷰어) |
| `E` | 파일 편집 (내부 에디터) |
| `F` | 파일 검색 (현재 디렉토리 하위 재귀) |
| `H` | 숨김 파일 표시 토글 |
| `?` | 도움말 (전체 화면, 스크롤 가능) |
| `Q` / `F10` | 종료 |
| `Ctrl+C` | 종료 |

### 입력 모드 단축키

파일 이름 변경(R), 새 폴더(K), 새 파일(N), 복사(C), 이동(M) 시 입력 프롬프트가 표시됩니다.

| 키 | 기능 |
|---|------|
| `←` `→` | 커서 좌/우 이동 |
| `Home` / `End` | 입력 시작/끝으로 이동 |
| `Delete` | 커서 위치 문자 삭제 |
| `Backspace` | 커서 앞 문자 삭제 |
| `Enter` | 입력 확인 |
| `Esc` | 입력 취소 |

### 뷰어 모드 단축키

V 키로 텍스트 파일을 열면 전체 화면 뷰어로 전환됩니다.

| 키 | 기능 |
|---|------|
| `↑` `↓` | 한 줄 스크롤 |
| `PageUp` / `PageDown` | 페이지 단위 스크롤 |
| `Home` / `End` | 파일 처음/끝으로 이동 |
| `/` | 텍스트 검색 (검색어 입력 후 Enter) |
| `n` | 다음 검색 매치로 이동 |
| `N` | 이전 검색 매치로 이동 |
| `Q` / `Esc` | 뷰어 닫기 |

- 10MB 초과 파일은 열 수 없습니다
- 바이너리 파일은 자동 감지되어 열리지 않습니다
- 검색 매치는 노란색 배경, 현재 매치는 노란색 반전으로 강조됩니다

### 에디터 모드 단축키

E 키로 텍스트 파일을 열면 전체 화면 에디터로 전환됩니다.

| 키 | 기능 |
|---|------|
| `↑` `↓` | 커서 위/아래 이동 |
| `←` `→` | 커서 좌/우 이동 (줄 경계 시 줄 이동) |
| `Home` / `End` | 줄의 처음/끝으로 이동 |
| `PageUp` / `PageDown` | 페이지 단위 이동 |
| `Backspace` | 커서 앞 문자 삭제 (줄 시작이면 윗줄과 병합) |
| `Delete` | 커서 위치 문자 삭제 (줄 끝이면 아랫줄과 병합) |
| `Enter` | 현재 위치에서 줄 분할 |
| `Ctrl+S` | 파일 저장 |
| `Esc` | 에디터 닫기 (수정 시 저장 확인) |

- 10MB 초과 파일은 열 수 없습니다
- 바이너리 파일은 자동 감지되어 열리지 않습니다
- 수정된 파일은 타이틀에 `[*]` 표시됩니다
- 한글 등 멀티바이트 문자 편집을 지원합니다

### 파일 검색 패턴 예제

F 키를 누르면 검색 패턴을 입력할 수 있습니다. 현재 디렉토리 하위를 재귀적으로 탐색합니다.

| 패턴 | 설명 |
|------|------|
| `*.rs` | 모든 Rust 소스 파일 |
| `test*` | "test"로 시작하는 파일 |
| `*.tar.gz` | 모든 tar.gz 압축 파일 |
| `README?` | README + 임의 한 글자 (README1, READMEa 등) |
| `config` | 파일명에 "config"가 포함된 파일 (와일드카드 없으면 부분 일치) |

- 패턴은 대소문자를 구분하지 않습니다
- 와일드카드 없이 입력하면 파일명에 해당 문자열이 포함된 파일을 모두 찾습니다
- 검색 결과 목록에서 `Enter`로 해당 파일 위치로 이동, `Backspace`로 원래 위치 복귀

### 컬럼 레이아웃

| 터미널 너비 | 컬럼 수 |
|------------|--------|
| 80 미만 | 1컬럼 |
| 80 ~ 119 | 2컬럼 |
| 120 이상 | 3컬럼 |

## UI 구성

### 화면 레이아웃

```text
┌─ /home/user/documents ──────────────────────────────┐
│ ..              <DIR> │ file_a.txt       1K │        │
│ src/            <DIR> │ file_b.rs      512B │        │
│ docs/           <DIR> │ archive.tar.gz  2M │        │
│ README.md        3K  │ link -> target   0B │        │
└─────────────────────────────────────────────────────┘
 README.md │ 3K │ 2025-03-01 │ rw-r--r-- [2개 선택]    ← 상태바
 Dir:3 File:4 │ 디스크: 120G/500G (24%) │ ? V E F N C M D R K Q  ← 정보바
```

화면은 3개 영역으로 구성됩니다:

1. **파일 목록 영역** - 멀티 컬럼으로 파일/디렉토리 표시, 타이틀에 현재 경로 표시
2. **상태바** - 커서 위치 파일의 이름, 크기, 수정일, 권한(rwxrwxrwx), 선택 개수
3. **정보바** - 디렉토리/파일 수, 디스크 사용량, 단축키 힌트 (에러 발생 시 에러 메시지 표시)

### 파일 타입별 색상

| 파일 타입 | 색상 | 예시 |
| --------- | ---- | ---- |
| 디렉토리 | **하늘색 (Cyan, 굵게)** | `src/`, `docs/` |
| 심볼릭 링크 | 보라색 (Magenta) | `link -> target` |
| 압축 파일 | 노란색 (Yellow) | `.tar.gz`, `.zip`, `.rar`, `.7z` 등 |
| 실행 파일 | 녹색 (Green) | 실행 권한이 있는 파일 |
| 일반 파일 | 흰색 (White) | `.txt`, `.rs`, `.md` 등 |

**스타일 우선순위**: 커서 위치(흰 배경) > 선택된 파일(노란색 굵게) > 파일 타입별 색상

### 모드별 하단바 표시

| 모드 | 하단바 | 설명 |
| ---- | ------ | ---- |
| Normal | 정보바 (회색) | 파일 수, 디스크 사용량, 단축키 힌트 |
| Input | 입력 프롬프트 (파란 배경) | 프롬프트 텍스트 + 블록 커서로 입력 위치 표시 |
| Confirm | 확인 프롬프트 (빨간 배경) | 삭제 등 위험 작업 확인 메시지 |
| Viewer | 뷰어 화면 | 파일 내용 + 줄 번호 + 하단 스크롤/검색 정보 |
| Editor | 에디터 화면 | 파일 편집 + 줄 번호 + 커서 + 하단 위치/저장 정보 |
| Help | 도움말 화면 | 전체 단축키 및 사용법 안내, 스크롤 가능 |
